## V. 核心设计原则

### 中核官方风格特征

1. **世界地图点阵背景**: 封面和章节页使用淡雅的点状世界地图纹理，体现国际化视野
2. **CNNC品牌标识**: 右上角或底部固定位置放置CNNC Logo，保持品牌一致性
3. **简洁几何装饰**: 使用简洁的几何线条和色块作为装饰，避免过度设计
4. **专业配色**: 以深蓝为主色调，传达专业、可靠、科技的视觉感受

### 设计细节规范

1. **渐变运用**: 深蓝到标准蓝的线性渐变用于封面和章节页背景
2. **透明度层次**: 装饰元素使用不同透明度创造层次感和呼吸感
3. **圆角处理**: 内容容器使用适度圆角 (`rx="8"`) 柔化视觉
4. **装饰线条**: 标题前使用竖向色块或横线作为引导装饰

## VII. 页面类型说明

### 1. 封面页 (01_cover.svg)

- **布局**: 左右分栏或叠加式布局
- **背景**: 左侧/顶部大面积深蓝色渐变 + 点状世界地图纹理
- **装饰**: 底部波浪曲线增加动感
- **标题**: 左对齐大号白色文字 + 副标题
- **图片**: 右侧建筑照片或全幅背景图
- **元素**: CNNC Logo（右上角）、日期、密级标识

### 2. 目录页 (02_toc.svg)

- **布局**: 左右分栏
- **左侧**: 深蓝色侧边栏 + 大字"目录"
- **右侧**: 列表式条目 + 序号 + 引导线
- **装饰**: 干净的分割线，保持视觉透气

### 3. 章节页 (02_chapter.svg)

- **背景**: 全屏深蓝色渐变 (`#002E5D` -> `#005BB7`)
- **中央**: 居中对齐大号章节数字 + 粗体章节标题
- **装饰**: 点状世界地图纹理 + 几何线条点缀
- **Logo**: 右上角CNNC Logo

### 4. 内容页 (03_content.svg)

- **顶部**: 极简标题栏 + 左上角蓝色矩形装饰
- **背景**: 纯白/极浅灰
- **内容**: 默认包含圆角虚线容器 (`stroke-dasharray="8,8"`)
- **底部**: 小字号页码 + 密级标识
- **右上角**: 小尺寸CNNC Logo

### 5. 结束页 (04_ending.svg)

- **背景**: 深蓝色渐变呼应章节页
- **中央**: "汇报完毕"致谢语
- **装饰**: 底部波浪线形成首尾呼应
- **元素**: 单位名称 + CNNC Logo

## IX. SVG技术约束

### 必须遵守规则

1. viewBox: `0 0 1280 720`
2. 使用 `<rect>` 元素绘制背景
3. 使用 `<tspan>` 实现文字换行（**禁止使用 `<foreignObject>`**）
4. 使用 `fill-opacity` / `stroke-opacity` 实现透明度
5. 禁止: `clipPath`（除非必需）、`mask`、`<style>`、`class`、`foreignObject`
6. 禁止: `textPath`、`animate*`、`script`
7. 渐变定义在 `<defs>` 中

## XI. 使用说明

1. 本模板专为中国核动力研究设计院及中核集团下属单位定制
2. 封面和章节页使用深蓝色渐变 + 世界地图点阵纹理背景
3. 内容页保持简洁白色背景，突出内容本身
4. 所有页面均需保留CNNC Logo以维持品牌一致性
5. 配色以深蓝色系为主，可根据具体内容适当调整强调色
